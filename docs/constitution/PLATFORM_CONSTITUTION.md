## Platform Test

### Before Introducing a Feature, Ask:

### Would This Capability Be Useful For:

- Personal trainers
- Therapists
- Music teachers
- Tattoo artists
- Tutors

### If Yes:

#### Platform Layer

### If No:

#### Application Layer

## Data Model Test

### Never Create a Specialized Entity Before Evaluating Whether It Can Be Represented As:

- User
- Organization
- Community
- Relationship
- Activity
- Offer
- Transaction
- Conversation
- Referral
- Affiliate
- Event

### Prefer Platform Primitives

### Avoid Vertical-Specific Entities Whenever Possible
